¿Cómo usar tu Nota de Crédito?
  1. Selecciona las joyas que deseas comprar y agrégalas al carrito.
  2. Durante el proceso de pago, busca el campo "Código de Nota de Crédito".
  3. Ingresa el código de tu Nota de Crédito (ej: NC-PE000020-ABCD).
  4. El sistema validará el código y aplicará el descuento automáticamente.
  5. ¡Listo! El monto se descontará de tu saldo disponible.
